# 🎓 Student Record Management System (C Language)

## 📌 Project Overview

The **Student Record Management System** is a menu-driven console application developed using the **C Programming Language**.

This project allows users to manage student information efficiently using **Structures** and **File Handling** to store records permanently.

The system performs complete CRUD operations:

* Create (Add)
* Read (Display/Search)
* Update
* Delete

---

## 🚀 Features

✅ Add Student Record
✅ Display All Records
✅ Search Student by ID
✅ Update Student Details
✅ Delete Student Record
✅ Permanent Data Storage using Files
✅ Menu Driven Interface

---
